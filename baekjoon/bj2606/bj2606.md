# 바이러스

- [문제 보기](https://www.acmicpc.net/problem/2606)
- 최근 학교에서 배운 Graph 자료 구조와 DFS(Depth First Search) 기법을 이용해 문제를 해결했다.
- 입력 받은 데이터를 Graph 자료 구조로 나타낸다. `Vertex` 구조체와 `IncidentEdge` 구조체를 만들어, 특정 `Vertex`에 연결된 모든 정점을 `IncidentEdge` 구조체를 담는 리스트로 저장시켰다.
- DFS는 함수의 재귀 호출로 구현했고, 해당 함수는 자신을 포함한 그래프의 크기를 반환하는 식으로 더 이상 방문할 노드가 없을 때까지 깊이 탐색한다.

## 다른 식의 구현

- 2차원 배열로 incident edge를 표현할 수 있고, 이는 메모리를 더 절약할 수 있다
- 꼭 graph 자료 구조를 이용하지 않더라도 edge가 하나씩 입력될 때마다 컴퓨터 1과 연관됐다면 숫자를 카운트하는 식으로 문제를 해결할 수도 있다.
